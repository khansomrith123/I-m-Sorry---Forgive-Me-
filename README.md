# Romantic Apology Website

A beautiful interactive apology website with animated hearts, music player, photo gallery, and playful interactions.

## Features

- 💌 Animated envelope with 3D flip reveal
- 💕 Floating hearts background animation
- ⏰ Live countdown timer since the mistake
- 🎵 Custom music player with volume control
- 📸 Polaroid-style photo gallery
- 🏃 Running "NO" button (playful interaction)
- 🎉 Celebration effects on forgiveness
- 📱 Fully responsive design

## Quick Start

1. Download `index.html`
2. Add your music file as `your-romantic-song.mp3`
3. Replace photo URLs with your own images
4. Open `index.html` in browser
